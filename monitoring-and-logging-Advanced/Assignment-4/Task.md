Assignment 4: Alerting in Prometheus
Objective: Configure alerting rules in Prometheus and test them using Alertmanager.

Steps:

Set up Prometheus Alertmanager in the k3s cluster.

Define alerting rules (e.g., high CPU usage, pod failures).

Test the alerting mechanism by simulating conditions that trigger alerts.

Deliverables:

Alerting rules configuration file.

Report documenting the setup and test results, including screenshots of received alerts.
