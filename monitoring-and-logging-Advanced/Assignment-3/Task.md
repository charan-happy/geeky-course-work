Assignment 3: Monitoring with Prometheus and Grafana
Objective: Deploy Prometheus and Grafana for monitoring a k3s cluster.

Steps:

Install Prometheus in the k3s cluster and configure it to scrape metrics from cluster components.

Deploy Grafana and integrate it with Prometheus as a data source.

Create a custom dashboard in Grafana to monitor node and pod metrics.

Deliverables:

Prometheus and Grafana YAML manifests.

A report with screenshots of the custom Grafana dashboard.
